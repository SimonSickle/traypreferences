Sample about for the TrayPreferences library sample app.


Features in this version:

• Replacement for Preferences UI elements utilizing the tray content provider.
• Utilizes tray content provider to replace SharedPreferences.
• Using tray as the content provider enables having multiprocess preferences.


Note:

• Current version only supported on Android 6.0 (API 23)


Get the source code at https://github.com/draekko-rand/traypreferences
